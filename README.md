## Angular Cli
### 安装Angular Cli工具
```
npm install -g @angular/cli@latest
```

### 若已经安装了低版本的angular-cli工具，执行：
```
npm uninstall -g angular-cli
npm cache clean -f
npm install -g @angular/cli@latest
```

## Json Server 
### 安装Json Server
```jshelllanguage
npm install -g json-server
```

### 运行Json Server
```
json-server --watch json/search.json
```
